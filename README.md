# Logistics & Supply Chain Warehouse

## Overview
Track shipments, costs, and delivery KPIs for logistics operations using Azure Synapse Analytics.

## Technologies 
- Azure Synapse Analytics
- Azure Data Factory
- Power BI
- SQL
- Python 3.9+

## Architecture
- Source: TMS, WMS, ERP systems
- ETL: Azure Data Factory
- Warehouse: Synapse Analytics
- Visualization: Power BI

## Features
- Shipment tracking
- Delivery performance metrics
- Cost analysis
- Route optimization
- Carrier performance
- Inventory in transit
- Real-time dashboards
